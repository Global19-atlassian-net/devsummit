Chrome Dev Summit is a two-day annual gathering where top web developers hear about the latest Web developments and get a glimpse into the future of the platform. The event will focus on:

  • Open web APIs and features that make the Web more powerful across verticals
  • Performance and tooling efforts that aim to make web development easier
  • Cutting-edge technologies that are changing the face of web development

During the two-day summit, we will focus on the open web platform with the goal of helping developers and businesses become successful on the Web by using open web standards and Google products.
