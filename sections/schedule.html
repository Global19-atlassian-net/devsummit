Monday, November 12th & Tuesday, November 13th

Chrome Dev Summit is a two day annual gathering of top web developers where they hear about the latest developments on the web and get a glimpse into the future of the platform. The event will focus on open web APIs and features that are making the web more powerful across verticals, performance and tooling efforts that aim to make web development easier and cutting edge technologies that are changing the face of web development.

Over the two days, we will focus on the open web platform with the goal to help developers and businesses become successful on the web using open web standards as well as Google products.
